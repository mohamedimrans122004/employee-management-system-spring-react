## Employee Management System

The **Employee Management System** is a full-stack web application designed to manage employee data efficiently within an organization. It provides complete CRUD (Create, Read, Update, Delete) functionality for employee profiles, using **Spring Boot** for the backend, **React** for the frontend, and **MySQL** for persistent data storage.
